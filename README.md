# eCabs
## Start with maven
1. Build parent project using __mvn install__ 
2. Start RabbitMq
3. start consumer and producer ./mvnw spring-boot:run

## Start with docker-compose 
1. Build parent project using __mvn install__ 
2. use docker-compose

## Endpoints
http://localhost:8081/booking 
(POST) - for create
(PUT) - for update
(DELETE) - for delete
